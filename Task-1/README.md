# Resources and Assumptions 

## In my website, I have utilized the following tools and techniques to ensure a smooth and user-friendly experience:

- > Intl-tel-input plugin: To provide a drop-down list of country codes for easy selection.
- > Media query: To make the website responsive and adaptable to various screen sizes.
- > Local storage: To store user input data and make it accessible across multiple sessions.
- > HTML, CSS, and JavaScript: To build and style the website effectively.
